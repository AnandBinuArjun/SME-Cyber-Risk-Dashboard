# Additional Platform Features

This document expands the **SME Cyber Risk Dashboard** scope with realistic platform features that improve operational depth, usability, collaboration, and governance without pushing the project into unrealistic enterprise complexity.

## 1. User Invitation and Onboarding
- Invite team members by email
- Accept invitation flow
- First-time onboarding wizard
- Guided setup for the first assessment
- Onboarding checklist

## 2. Multi-Domain Support
- Allow one organization to manage multiple domains
- Support primary and secondary domains
- Label domain ownership and purpose
- Maintain separate assessment history per domain

This is realistic for SMEs that may operate:
- a main website
- a mail domain
- a support domain
- regional or campaign-specific domains

## 3. Asset Inventory
- Maintain a list of business assets
- Track website assets
- Track email domains
- Track admin portals
- Track critical external services
- Assign asset criticality levels

This improves realism because findings and risk priority should depend on asset importance.

## 4. Domain Ownership Verification
- Verify user control over a domain before assessment
- DNS TXT verification
- Email-based verification
- Manual admin approval flow

This adds credibility and supports ethical assessment controls.

## 5. Scheduled Assessments
- Monthly reassessment scheduling
- Quarterly posture reviews
- Reminder before reassessment
- Recurring assessment history

This helps position the platform as a continuous monitoring tool rather than a one-time assessment application.

## 6. Scan Queue and Status Tracking
- Queued
- Running
- Completed
- Failed
- Cancelled

This makes the platform feel more operational and realistic.

## 7. Evidence Viewer
- Raw DNS response display
- Raw security header output
- TLS certificate details
- Questionnaire answer evidence
- Timestamped evidence panel

This improves transparency, traceability, and trust.

## 8. Analyst Notes and Comments
- Add notes to findings
- Internal collaboration comments
- Remediation discussion thread
- Analyst observations per assessment

This is especially useful when multiple users collaborate inside the same organization.

## 9. Finding Assignment
- Assign findings to team members
- Assign recommendations to responsible users
- Add remediation due dates
- Track ownership

This makes the system feel like a working remediation platform rather than just a report generator.

## 10. Remediation Workflow
- Open
- Assigned
- In Progress
- Resolved
- Validated
- Accepted Risk

This is stronger and more realistic than simply showing recommendations.

## 11. Risk Acceptance Workflow
- Mark a finding as accepted risk
- Record the reason for acceptance
- Set an expiry date for risk acceptance
- Record who approved the acceptance

This is a strong real-world governance feature for academic presentation.

## 12. Reassessment Comparison
- Compare two assessments side by side
- Show fixed findings
- Show new findings
- Show worsened categories
- Show improved categories

This is one of the most valuable additions because it demonstrates change over time.

## 13. Priority-Based Remediation Roadmap
- Immediate actions
- 30-day plan
- 60-day plan
- 90-day plan

This makes recommendations more actionable and business-friendly.

## 14. Security Posture Timeline
- Timeline of assessments
- Major posture changes
- Report generation history
- Remediation milestones

This adds value to both the analytics layer and the user experience.

## 15. Dashboard Personalization
- Widgets by role
- User-selected visible cards
- Saved filters
- Preferred dashboard views

Useful, but lower priority than core security and assessment logic.

## 16. Search and Advanced Filters
- Search findings by keyword
- Filter by severity
- Filter by category
- Filter by status
- Filter by assessment date
- Filter by domain

This is a basic but highly necessary usability feature.

## 17. Export Options
- Export findings as CSV
- Export recommendations as CSV
- Export executive summary as PDF
- Export full assessment report

These features improve practical usability for both academic demonstrations and stakeholder use.

## 18. Notification Center
- Scan completion alerts
- Certificate expiry alerts
- High-risk finding alerts
- Overdue remediation alerts
- New report generated alerts

This adds operational realism quickly.

## 19. Security Knowledge Tips
- "Why this matters" explanations for issues
- Short awareness tips
- Security glossary tooltips
- Contextual education panel

This helps non-technical SME owners understand the significance of findings.

## 20. Business Risk Categories
Map technical issues into business-oriented categories such as:
- Fraud risk
- Downtime risk
- Reputational risk
- Compliance risk
- Data loss risk

This is a strong feature because it bridges technical reporting with business decision-making.

## 21. Readiness Scores by Theme
- Phishing readiness
- Ransomware readiness
- Access control readiness
- Recovery readiness
- Governance readiness

These sub-scores strengthen the analytics and explainability of the platform.

## 22. Platform Activity Logs
- Who logged in
- Who created assessments
- Who updated remediation status
- Who downloaded reports
- Who changed user roles

This reinforces the administrative and audit side of the project.

## 23. Report Versioning
- Keep versions of generated reports
- Track report generation dates
- Compare old and new versions
- Maintain audit traceability

This is a mature and practical reporting feature.

## 24. Secure Settings Management
- Change password
- Manage profile
- Update notification preferences
- Manage organization settings
- View active sessions or device visibility

This makes the platform feel more complete and user-centered.

## 25. Dark Mode / Light Mode
- Dark mode as the default experience
- Optional light mode toggle

This is not a core cybersecurity feature, but it improves usability and presentation quality.

## Best Features to Prioritize from This Set
If you want the strongest additions from this list without expanding scope too far, prioritize:
- Multi-domain support
- Asset inventory
- Domain ownership verification
- Scheduled assessments
- Scan queue and status tracking
- Evidence viewer
- Finding assignment
- Remediation workflow
- Reassessment comparison
- Priority-based remediation roadmap
- Notification center
- Platform activity logs
